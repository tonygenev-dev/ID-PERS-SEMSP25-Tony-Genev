# Terminal-Based Pirate Adventure Game
Overview
This is a real-time, terminal-based adventure game developed in C++ using Object-Oriented Programming (OOP) principles. The project utilizes the ftxui library to create a functional and interactive terminal user interface, bringing a classic pirate-themed experience to the modern command line.


The core of this project was to bridge the gap between complex software logic and an engaging user experience, focusing on real-time state management and efficient hardware-software interaction within the terminal environment.


Key Features
Real-Time Gameplay: Engineered a responsive game loop that handles user input and world events simultaneously.

Object-Oriented Architecture: Utilized OOP to manage game entities such as ships, islands, and players, ensuring clean, modular, and maintainable code.


Terminal UI (ftxui): Implemented a sophisticated layout using the ftxui library to create a visually appealing and intuitive interface directly in the terminal.

Dynamic Interaction: Features a system where players can navigate and interact with a procedural or pre-defined pirate world.

Technical Stack
Language: C++ 


Paradigm: Object-Oriented Programming (OOP) 


Library: ftxui (Functional Terminal User Interface) 

Tools: Git for version control 

How to Build & Run
Clone the repository:

Bash
git clone 
Install Dependencies: Ensure you have a C++ compiler (GCC/Clang) and the ftxui library installed.

Compile:

Bash
mkdir build && cd build
cmake ..
make
Launch:

Bash
./PirateAdventure
